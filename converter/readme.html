How To Build

If you're using Windows, simply open the vcxproj file in this folder.

For Linux/X11, you'll need to use the supplied WAF script. If you don't have WAF, you can download it from Google Code. Or, you can type this into a python script: 
import urllib
import os
import stat
urllib.urlretrieve("http://waf.googlecode.com/files/waf-1.6.2", "waf")
os.chmod('waf', stat.S_IEXEC)
Next, type this: 
$ ./waf configure build run
